# 反应速率影响

+ 内部因素
  + 反应物状态: 固/液/气
  + 反应难易程度: 条件苛刻/K
+ 外部因素
  + 催化剂
  + 压强
  + **温度**: **温度升高，正逆反应速率均升高**
    + p.s. 方程式 A=2B 则 (vA/vB) = 1/2

# 反应**平衡的条件**
+ 速率**按照反应系数比!**
  + v正=v逆
  + 断/成键速率
  + 生成/消耗反应/生成物的速率
+ 数据计算(恒TP/恒TV)
  + 物质的量
  + 物质反应速率
  + 压强(恒V)
  + 转化率
  + 密度
  + 平均相对分子质量
  + 颜色
